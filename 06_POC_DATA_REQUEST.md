# Voltix PoC 자료 요청서

- 문서명: PoC 자료 요청서
- 목적: 전기도면 기반 수량산출 자동화 MVP 검증에 필요한 샘플 자료 확보
- 작성일: 2026-07-09
- 버전: v0.1 Draft

---

## 1. 요청 배경

Voltix MVP는 DWG/DXF 전기도면에서 주요 전기 심볼을 추출하고, 심볼을 실제 전기자재 품목과 매핑하여 수량표, 견적서, 발주 리스트까지 생성하는 것을 목표로 한다.

PoC의 핵심은 실제 현장 도면 기준으로 자동 인식률, 검수 후 정확도, 수작업 대비 시간 절감 효과를 검증하는 것이다. 이를 위해 실제 도면, 수작업 산출표, 자재 기준표, 단가표가 필요하다.

---

## 2. 필수 요청 자료

| 구분 | 요청 자료 | 권장 수량 | 필수 여부 | 비고 |
|---|---|---:|---|---|
| 도면 | 실제 DWG 전기도면 | 5~10개 | 필수 | 세대 전기 평면도 우선 |
| 도면 | DXF 변환 파일 | 가능 시 동일 수량 | 권장 | DWG 파싱 검증 보조 |
| 산출표 | 수작업 수량산출 엑셀 | 1~3개 | 필수 | 자동 산출 결과 비교 기준 |
| 자재표 | 주요 전기자재 품목표 | 1개 | 필수 | 품명, 규격, 단위 포함 |
| 품번표 | 제조사 품번표 | 1개 | 권장 | 제조사, 품번, 규격 포함 |
| 단가표 | 자재 단가표 샘플 | 1개 | 권장 | 기준가, 공급가, 프로젝트가 구분 가능하면 좋음 |
| 기준자료 | 심볼 범례 또는 도면 Legend | 가능 시 | 권장 | 블록명·레이어 해석에 유용 |
| 검증자 | PoC 검토 담당자 | 1명 이상 | 필수 | 결과 검수와 피드백 담당 |

---

## 3. 도면 자료 요청 기준

### 3.1 우선 확보 도면

1. 공동주택 세대 전기 평면도
2. 오피스텔 세대 전기 평면도
3. 반복 배선기구가 많은 도면
4. 블록명과 레이어명이 비교적 정리된 DWG
5. 수작업 산출표가 함께 있는 도면

### 3.2 후순위 도면

1. 스캔 PDF 도면
2. 이미지화된 도면
3. 건축·기계·소방 등 타 공종 중심 도면
4. 블록 정보가 모두 깨져 있거나 레이어 기준이 없는 도면
5. 수작업 산출 기준을 확인할 수 없는 도면

## 4. MVP 대상 심볼 자료

아래 심볼이 포함된 도면을 우선 요청한다.

| 심볼 분류 | 예시 | 요청 이유 |
|---|---|---|
| 스위치 | 1구, 2구, 3구, 조광, 방우형 | 반복 수량산출 효과가 큼 |
| 콘센트 | 일반, 접지, 방우, 전용, 에어컨 | 자재 품번 매핑 필요성이 큼 |
| 통신 인출구 | LAN, TEL, TV | 도면 표기 차이가 커 검증 필요 |
| 통합수구 | 통신 통합 아웃렛 | 실제 자재 매핑 검증 필요 |
| 세대분전반 | PANEL, 분전반 | 수량은 적지만 견적 영향이 큼 |

## 5. 수작업 산출표 요청 형식

가능하면 다음 컬럼을 포함한 엑셀 자료를 요청한다.

| 컬럼 | 설명 | 필수 여부 |
|---|---|---|
| 프로젝트명 | 현장 또는 프로젝트명 | 권장 |
| 도면명 | 산출 기준 도면명 | 필수 |
| 세대 타입 | 59A, 84A 등 | 권장 |
| 층/구역 | 층 또는 구역 정보 | 권장 |
| 자재 분류 | 스위치, 콘센트 등 | 필수 |
| 품명 | 자재명 | 필수 |
| 규격 | 자재 규격 | 권장 |
| 품번 | 제조사 품번 | 권장 |
| 단위 | EA, SET 등 | 필수 |
| 수량 | 수작업 확정 수량 | 필수 |
| 비고 | 산출 기준 또는 메모 | 권장 |

---

## 6. 보안 및 익명화 요청

자료 반출 또는 공유가 제한되는 경우 다음 방식으로 처리한다.

1. 현장명, 고객사명, 주소 등 민감 정보 삭제
2. 도면 타이틀 블록 내 고객 정보 마스킹
3. 파일명 익명화
4. 외부 공유 불가 시 내부 보안 PC 또는 원격 환경에서 검증
5. 자료 사용 범위를 PoC 검증 목적으로 제한

---

## 7. 검증 담당자에게 요청할 피드백

| 항목 | 확인 내용 |
|---|---|
| 자동 인식 결과 | 실제 심볼과 자동 분류 결과가 맞는지 |
| 수량 결과 | 수작업 산출표와 자동 산출표가 얼마나 일치하는지 |
| 미분류 심볼 | 사람이 보면 어떤 자재인지 알 수 있는지 |
| 중복 의심 | 실제 중복인지, 도면상 필요한 반복 표기인지 |
| 검수 UI | 도면 위에서 수정하는 흐름이 실무에 맞는지 |
| 견적 연결 | 산출 수량이 견적·발주에 바로 쓸 수 있는지 |

## 8. 자료 확보 완료 기준

PoC 자료 확보 단계는 다음 조건을 만족하면 완료로 본다.

1. DWG 또는 DXF 샘플 도면 5개 이상 확보
2. 수작업 산출 엑셀 1개 이상 확보
3. 주요 자재 품목표 1개 확보
4. 제조사 품번표 또는 단가표 중 1개 이상 확보
5. PoC 검증 담당자 1명 이상 지정
6. 자료 사용 범위와 보안 조건 확인

## 9. 요청 메시지 초안

아래 문구를 자료 요청 메일 또는 메신저에 사용할 수 있다.

```text
Voltix 전기자재 수량산출 자동화 PoC 검증을 위해 실제 전기도면과 수작업 산출 자료가 필요합니다.

가능하시다면 공동주택 또는 오피스텔 세대 전기 평면도 DWG 파일 5~10개, 해당 도면 기준 수작업 산출 엑셀 1개 이상, 주요 자재 품목표 또는 품번표를 공유 부탁드립니다.

자료 내 현장명, 고객사명, 주소 등 민감 정보는 삭제 또는 마스킹해도 됩니다. 공유 자료는 PoC 검증 목적의 자동 인식률, 수량 정확도, 업무 시간 절감 효과 확인에만 사용하겠습니다.
```
